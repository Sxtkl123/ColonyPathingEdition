本mod旨在优化殖民地游戏体验，让玩家能够不被各种奇怪的AI问题打扰城建的乐趣。

首先，我们让市民寻路更听从玩家的指挥，可以让玩家通过道路基建优化殖民地内的交通，同时让寻路系统能够更好的适应复杂的地形。除此之外，顺带解决一些minecolonies寻路系统中的陈年bug。
目前还在逐步优化中，现在的效果可能只是差强人意，相信后面在大家的帮助下能够变得更好。

其次，我们优化了市民进食AI，让岗位有食物的工人不再频繁前往餐厅，同时增强了餐厅的服务与烧制效率，并优化了厨房的烧制效率，从而实现食物的供给平衡。

修改了生病AI，让市民能够更快自愈，同时增强医院的治疗能力，并让医生同时具备“军医”的特质。

烧制AI的修改同时也辐射到了冶石工，现在冶石工也会更好的分工合作，加速石材冶炼进程。

除此之外，我们还优化了殖民地的建筑工(代码由sxtkl实现)，伐木工等多个工人的工作效率，具体内容请见curseforge和bilibili的mod介绍。

目前处于爆发更新阶段！敬请期待！
